# Tatva → moved

This blog has merged into **[sumityadav.com.np](https://sumityadav.com.np)**.

`tatva.sumityadav.com.np` is now a redirector: every path (including old
`/posts/...` links) forwards to the same path on `sumityadav.com.np`.

- Served from `docs/` (GitHub Pages): `index.html` + `404.html` do a
  path-preserving redirect; `CNAME` keeps the `tatva.sumityadav.com.np` domain.
- The blog content and its builder now live in the
  [`dotehacker.github.io`](https://github.com/dotehacker/dotehacker.github.io) repo.

The old Jekyll-like source (posts, layouts, build.js) was removed from this
repo; it remains in git history.
